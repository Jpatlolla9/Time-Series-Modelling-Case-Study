# Time-Series-Modelling-Case-Study
# Electricity Load Forecasting

This project forecasts German electricity demand using statistical, machine learning and deep learning techniques. The analysis uses hourly electricity load data from Open Power System Data (2015–2020) and temperature data from the Open-Meteo Archive API. Models developed include Benchmark methods, SARIMA, SARIMAX, Random Forest, Gradient Boosting and LSTM.

The models were evaluated using MAE, RMSE, MAPE and MASE to compare forecasting performance. The results showed that advanced models outperformed the benchmark methods, with SARIMAX providing the best balance between forecasting accuracy, interpretability and practical implementation.
